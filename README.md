# 📱 Pokédex — React Native

Projet personnel pour apprendre les bases de **React Native** avec **Expo**.

Je suis la vidéo de [Grafikart — Découverte de React Native, création d'un Pokédex](https://www.youtube.com/watch?v=0EOucUY_F0c) comme support d'apprentissage.

## 🎯 Objectif

Construire un Pokédex fonctionnel pour pratiquer les fondamentaux :

## 🛠️ Stack technique

| Outil | Rôle |
|---|---|
| [React Native](https://reactnative.dev/) | Framework mobile |
| [Expo](https://expo.dev/) | Toolchain & runtime |
| [Expo Router](https://docs.expo.dev/router/introduction/) | Routing basé sur le filesystem |
| [TypeScript](https://www.typescriptlang.org/) | Typage statique |

## 🚀 Lancer le projet

1. Installer les dépendances

   ```bash
   pnpm install
   ```

2. Démarrer le serveur de développement

   ```bash
   pnpm run start
   ```

3. Scanner le QR code avec [Expo Go](https://expo.dev/go) (mobile) ou lancer un émulateur
